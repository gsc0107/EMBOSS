ehmmindex

Function

Description

ehmmindex is a utility that creates a binary SSI ("squid sequence index" format) index for an HMM database file called database. The new index file is named database.ssi. An SSI index file is required for ehmmfetch to work, and also for the PVM implementation of ehmmpfam.
