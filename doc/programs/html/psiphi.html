1.0 SUMMARY

Calculates the psi and phi torsion angles around the alpha carbons (CA) in (a specified stretch of) a specified chain of a protein structure from co-ordinates of the mainchain atoms in the two planes around it

2.0 INPUTS & OUTPUTS

PSIPHI reads in a protein CCF file (clean coordinate file---not PDB format) containing the chain for which torsion angles are to be calculated. The user can specify the chain and the stretch of residues for which psi and phi angles are to be calculated.
