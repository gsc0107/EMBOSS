
                              EMBOSS: profgen
     _________________________________________________________________
   
                                Program profgen
                                       
Function

   Generates various profiles for each alignment in a directory
   
Description

   This is part of Jon Ison's protein structure analysis package.
   
   This package is still being developed.
   
   Please ignore this program until further details can be documented.
   
   All further queries should go to Jon Ison. (Jon Ison)
   
Usage

   Here is a sample session with profgen:

% profgen

Command line arguments

   Mandatory qualifiers (* if not always prompted):
   -infpath            string     Directory containing the extended alignments
                                  (input)
   -infextn            string     File extention of extended alignment files
                                  (input)
   -type               menu       Select type
*  -threshold          integer    Enter threshold reporting percentage
*  -datafile           matrixf    Scoring matrix
*  -open               float      Gap opening penalty
*  -extension          float      Gap extension penalty
*  -smpfpath           string     Directory for generated simple profiles
                                  (output)
*  -smpfextn           string     File extention of generated simple profile
                                  (output)
*  -gbpfpath           string     Directory for generated Gribskov profiles
                                  (output)
*  -gbpfextn           string     File extention of generated Gribskov profile
                                  (output)
*  -hnpfpath           string     Directory for generated Henikoff profiles
                                  (output)
*  -hnpfextn           string     File extention of generated Henikoff profile
                                  (output)

   Optional qualifiers: (none)
   Advanced qualifiers: (none)
   General qualifiers:
  -help                bool       report command line options. More
                                  information on associated and general
                                  qualifiers can be found with -help -verbose
   

   Mandatory qualifiers Allowed values Default
   -infpath Directory containing the extended alignments (input) Any
   string is accepted ./
   -infextn File extention of extended alignment files (input) Any string
   is accepted .salign
   -type Select type
   F (Frequency)
   G (Gribskov)
   H (Henikoff)
   F
   -threshold Enter threshold reporting percentage Integer from 1 to 100
   75
   -datafile Scoring matrix Comparison matrix file in EMBOSS data path
   'Epprofile' for Gribskov type, or EBLOSUM62
   -open Gap opening penalty Any numeric value 3.0
   -extension Gap extension penalty Any numeric value 0.3
   -smpfpath Directory for generated simple profiles (output) Any string
   is accepted ./tmp/
   -smpfextn File extention of generated simple profile (output) Any
   string is accepted .freq
   -gbpfpath Directory for generated Gribskov profiles (output) Any
   string is accepted ./tmp/
   -gbpfextn File extention of generated Gribskov profile (output) Any
   string is accepted .gribs
   -hnpfpath Directory for generated Henikoff profiles (output) Any
   string is accepted ./tmp/
   -hnpfextn File extention of generated Henikoff profile (output) Any
   string is accepted .henik
   Optional qualifiers Allowed values Default
   (none)
   Advanced qualifiers Allowed values Default
   (none)
