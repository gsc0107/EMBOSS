About Jemboss

    Jemboss is currently being developed at the HGMP. It is an interface to EMBOSS, which can be run in one of two modes. The interface can be run with a local installation, this is the standalone mode. Alternatively, it may be run using a remote server, i.e. in a client-server mode.
