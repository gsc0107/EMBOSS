MYEMBOSS

The MYEMBOSS package is an empty EMBASSY package where users can safely build their own EMBOSS applications.

We also provide the MYEMBOSSDEMO package. This is an EMBASSY package where users can safely modify a set of demonstration applications and practise building their own EMBOSS applications. By keeping the demonstration applications separate users can run "make install" in the MYEMBOSS package and use MYEMBOSSDEMO for testing without installing.
