DOMALIGN Applications

The DOMALIGN programs were developed by Jon Ison and colleagues at HGMP for their protein domain research. They are included as an EMBASSY package as a work in progress.

Applications in the current domalign release

Program name Description
allversusall Sequence similarity data from all-versus-all comparison
domainalign Generate alignments (DAF file) for nodes in a DCF file
domainrep Reorder DCF file to identify representative structures
seqalign Extend alignments (DAF file) with sequences (DHF file)
