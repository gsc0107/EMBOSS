Function

Reports transmembrane helices

Description

Prediction of transmembrane helices in proteins.

Usage

Here is a sample session with etmhmm 


% etmhmm 
Reports transmembrane helices
Input (aligned) sequence set: 5h2a_crigr.fsa
Output file [5h2a_crigr.etmhmm]:
